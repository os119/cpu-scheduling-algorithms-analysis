# CPU Scheduling Algorithms Analysis

This repository includes my Python project on CPU scheduling algorithms.

I built this project to better understand how different scheduling policies affect process execution and overall system performance. The project mainly focuses on First Come First Served (FCFS), Shortest Job First (SJF), and Round Robin (RR).

Working on this helped me build a clearer understanding of core operating systems concepts, especially how scheduling decisions can affect waiting time, turnaround time, and fairness between processes.

## Main areas covered
- First Come First Served (FCFS)
- Shortest Job First (SJF)
- Round Robin (RR)
- Average waiting time and turnaround time
- Gantt chart visualization
